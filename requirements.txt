# ─────────────────────────────────────────────────────────────────────────────
# Dependencies for the METR-LA traffic-forecasting GNN.
#
# Colab note: Colab already ships a PyTorch build matched to its GPU/CUDA driver,
# and reinstalling torch can break that match. On Colab only the two PyG packages
# are needed:
#
#     !pip install -q torch_geometric torch-geometric-temporal
#
# These pins are for a local install, where torch isn't preinstalled.
# Versions are left loose on purpose: the PyG-Temporal loader internals shift
# between releases, so the safest recipe is "latest compatible". If a local
# install ever errors about a missing compiled op (torch_scatter / torch_sparse),
# see the fallback note in README.md.
# ─────────────────────────────────────────────────────────────────────────────

torch>=2.0          # the deep-learning engine (tensors, autograd, GPU)
torch-geometric>=2.3        # graph neural network primitives (PyG)
torch-geometric-temporal>=0.54   # spatio-temporal extensions: METR-LA loader, A3T-GCN
matplotlib>=3.5     # Stage 5: the predicted-vs-actual plot
numpy>=1.23         # array math / shape inspection
